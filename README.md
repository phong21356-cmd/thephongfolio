> [!IMPORTANT]
> Because the prototype HTML uses custom template strings (`{{ accent }}`, etc.) in inline styles, standard bundlers/compilers like Vite will throw syntax errors when processing CSS. We are serving this project using `http-server` which acts as a simple static file server.

## 🚀 Setup and Launch

### 1. Install Dependencies
```bash
npm install
```

### 2. Run local server
```bash
npm run dev
```

The app will start at:
👉 **[http://localhost:5173/](http://localhost:5173/)**
